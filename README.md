1. 매주 1명씩 문제 출제 : 순서는 아래 표에서 확인  
2. 출제자는 난이도 상 / 중급 이상 문제 1~2개 선정 : 백준 기준 골드 5 이상
3. 구성원은 90분 제한 시간 안에 풀이 : 디스코드 채널에서 진행
4. 스터디 후, 각자 리뷰(접근법 / 복기)
